# Local Docs Channel

## Purpose

本地文档通道属于 `local_docs`，用于补充在线协作平台之外的文档证据。

可映射的能力包括：

- `docs_fetch`
- `file_metadata`

## Typical Sources

本地文档通道可以来自：

- Markdown 文档
- 导出的 docx / txt / pdf
- 项目目录中的计划、复盘、会议纪要

## Evidence Guidance

本地文档通道适用于：

- 没有统一在线协作平台
- 有部分资料以导出文件形式保存
- 需要补充在线平台未覆盖的信息

归一化后，通常产生：

- `doc`
- `file_metadata`

## Collection Guidance

读取本地文档时，优先提炼：

- 文档标题
- 时间
- 关键结论
- 待办与计划
- 与工作主题的关联
