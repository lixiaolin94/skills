# Git Channel

## Purpose

Git 通道属于 `git`，用于提供代码推进证据。

可映射的能力包括：

- `git_history`

## Typical Sources

Git 通道可以来自：

- 本地 Git 仓库
- GitHub CLI / skill
- GitLab CLI / skill

## Evidence Guidance

Git 通道重点回答：

- 本周哪些项目有真实代码推进
- 提交主题集中在哪些方向
- 哪些代码工作能和协作证据互相印证

归一化后，通常产生：

- `commit`

## Collection Guidance

Git 采集的目标不是罗列所有提交，而是提炼：

- 仓库
- 主题
- 时间
- 与其他证据的关联

如果当前环境是本地仓库优先，可优先检查常见开发目录中的 Git 仓库。
